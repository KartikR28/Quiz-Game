# Python Quiz Game (API + Tkinter)

A desktop quiz application built using Python that fetches real-time questions from an external API and displays them in an interactive GUI.

---

## Features
- Real-time questions using REST API
- Clean graphical interface with Tkinter
- Score tracking and instant feedback
- Modular and readable code structure

---

## Tech Stack
- Python
- Tkinter
- Requests
- Open Trivia DB API

---

## Project Structure
- `main.py` – Entry point
- `data.py` – API integration
- `quiz_brain.py` – Quiz logic
- `question_model.py` – Data model
- `ui.py` – GUI implementation

---
## How to Run
-'bash'
pip install requests
python main.py
